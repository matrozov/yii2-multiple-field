# Yii2 Multiple field widget and nested fields validation

[![No Maintenance Intended](http://unmaintained.tech/badge.svg)](http://unmaintained.tech/)

Yii2 widget for handle multiple fields for an attribute of model and tabular input for batch of models.

---

# DEPRECATED
This repository is no longer maintained and will eventually be removed. The functionality has been split into two repositories:
* [Yii2 Multiple Field Widget](https://github.com/matrozov/yii2-multiple-field-widget) - widget implementing visual
  management of nested models
* [Yii2 Nested Field Validator](https://github.com/matrozov/yii2-nested-field-validation) - a set of validators for
  implementing nested model validation.

---

## Installation

```bash
php composer.phar require matrozov/yii2-multiple-field "dev-master"
```

or add

```
"matrozov/yii2-multiple-field": "dev-master"
```

to the require section of your `composer.json` file.
